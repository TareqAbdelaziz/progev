Configuration Twilio Sandbox (gratuit pour tester)
Twilio propose un sandbox WhatsApp gratuit. Chaque destinataire doit envoyer un code d'activation une seule fois depuis son téléphone.
Étape 1 — Créer ton compte Twilio
Gratuit · Aucune carte bancaire requise pour le sandbox
1
Va sur twilio.com → "Sign up" → vérifie ton email et ton numéro de téléphone.
2
Dans le dashboard Twilio, trouve ton Account SID et ton Auth Token sur la page d'accueil.
3
Va dans Messaging → Try it out → Send a WhatsApp message. Twilio te donne un numéro sandbox (ex: +14155238886) et un code (ex: join lamp-tiger).
4
Depuis ton téléphone WhatsApp, envoie le code au numéro sandbox. Fais pareil pour chaque destinataire (direction, chef chantier...).
